Кодировка в устаревшем теге "FONT"

#BruceWillis

Алгоритм следующий:
Заменяем все неправильные шестнадцатеричные символы нулями 0, #BruceWillis превращается в b00ce000000
“Добиваем” число нулями до количества знаков, делимого на 3, получаем b00c e000 0000
Разбиваем на три группы, каждая из которых отвечает за одну компоненту RGB: RGB (b00c, e000, 0000)
Обрезаем каждый из аргументов справа, оставляя 2 символа,
и получаем наш результат RGB (b0, e0, 00) = #B0E000 или он же RGB(176, 224, 0).

#CANDY

Аналогично:
#CANDY превращается в ca0d0
Далее, получаем ca0d 0000 0000
RGB (ca0d, 0000, 0000)
Результат RGB (ca, 00, 00) = #CA0000 или он же RGB(202, 0, 0).

#chucknorris

Аналогично:
#chucknorris превращается в c00c0000000
Далее, получаем c00c 0000 0000
RGB (c00c, 0000, 0000)
Результат RGB (c0, 00, 00) = #C00000 или он же RGB(192, 0, 0).
